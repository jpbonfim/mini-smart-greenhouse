2  (VDD)  → +5V
3  (V0)   → Potentiometer Center (Contrast)
4  (RS)   → Arduino D12
5  (RW)   → GND
6  (E)    → Arduino D11
7  (D0)   → Not connected
8  (D1)   → Not connected
9  (D2)   → Not connected
10 (D3)   → Not connected
11 (D4)   → Arduino D6
12 (D5)   → Arduino D5
13 (D6)   → Arduino D4
14 (D7)   → Arduino D3
15 (A)    → +5V through 220Ω resistor
16 (K)    → GND

══════════════════════════════════════════════════════════════════════════════════
                    ⚠️  VOLTAGE DIVIDER (CRITICAL!) ⚠️
══════════════════════════════════════════════════════════════════════════════════

    Arduino D9 (5V) ────[1K Resistor]────┬──── Bluetooth RX (3.3V)
                                         │
                                    [2K Resistor]
                                         │
                                        GND

    Math: Vout = 5V × (2K / (1K + 2K)) = 5V × 0.667 = 3.33V ✓


⚠️  WARNING: DO NOT connect Arduino D9 directly to Bluetooth RX!
    This will damage the Bluetooth module! Always use the voltage divider!

                                 TESTING STEPS

  1. ✓ Connect LCD and adjust contrast (should see boxes on display)
  2. ✓ Upload Arduino code and verify "Smart Greenhouse" appears

  5. ✓ Create RFCOMM binding (Linux) or note COM port (Windows)
  6. ✓ Update app.py with correct port
  7. ✓ Run Flask app: python3 app.py
  8. ✓ Open browser to http://localhost:5000
  9. ✓ Click preset buttons and verify LCD changes
 10. ✓ Check Serial Monitor for debug messages
